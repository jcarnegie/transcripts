# Run the Digital Avatar Coaching Client
### Install
```
add PINECONE_API_KEY and OPENAI_API_KEY to your environment variables
make install
```

### Run
```
python -m src.chat  # Run the chat agent directly
python -m src.main  # Run the webserver 
```
